# ⚽ Financial Asset Valuation

This project was carried out as part of the **Financial Asset Analysis** course in the M1 ECAP program.  
Its objective is to analyze the performance of **9 publicly listed football clubs** using various financial methods.

## 📑 Project Documentation
We produced a **detailed report** explaining our methodology and results.  
📄 **Access the document (in French)**: [Click here](https://docs.google.com/document/d/1kdnBFq-cdPv0nZX7j7HDvFyNhwSTjsbJxQFPWvh0vEU/edit?tab=t.0)

## 📌 Project Objectives
- 📊 Analyze the profitability and risk of selected financial assets.  
- 📈 Apply financial models to optimize portfolio management.  
- 🔍 Study the impact of sporting performance on the stock market valuation of football clubs.  

## 📂 Project Structure
📁 `data/` : Contains the financial data used.  
📁 `scripts/` : R scripts for data processing and analysis.  
📁 `results/` : Results of the analyses.
